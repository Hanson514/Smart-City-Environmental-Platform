# Smart City Environmental Protection Platform

## Project Overview

Smart City Environmental Protection Platform is a software engineering team project developed at the University of Birmingham as part of the Software Engineering module.

The project aims to support environmental monitoring, data management, and sustainable urban development through a modern full-stack web application.

## Backend Architecture

The backend follows a layered architecture:

Frontend (Node.js)

↓

REST API Layer (FastAPI)

↓

Service Layer

↓

Database Layer (Supabase / PostgreSQL)

This architecture improves scalability, maintainability, and separation of concerns.
